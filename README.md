# <アプリ名>
オンライン合コン掲載サイト
## サイト概要
オンライン合コンを掲載、登録したユーザーが参加できるサイト

### サイトテーマ
オンラインでの男女の出会いを斡旋

### テーマを選んだ理由
コロナで出会いが少なくなったこの時期にもっと人との出会いを活発化したい

### ターゲットユーザ
合コンや街コンに参加している男女

### 主な利用シーン
今まで合コンをしていた人たちが利用する
